# Apache Oozie (apache-oozie)
Apache Oozie is a workflow scheduler system to manage Apache Hadoop jobs. It allows users to define workflows as directed acyclic graphs of actions and supports coordination of recurring workflows based on time and data availability.

### Apache Oozie REST API
The Oozie Web Services API provides REST endpoints for submitting, managing, and monitoring workflow, coordinator, and bundle jobs, along with a Java client API and SLA management capabilities.
